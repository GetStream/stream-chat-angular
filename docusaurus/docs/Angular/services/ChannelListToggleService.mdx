# ChannelListToggleService

The `ChannelListToggleService` can be used to toggle the channel list.

**`Deprecated`**

This service can only be used with [theming-v1](../concepts/theming-and-css.mdx), if you are using [thmeing-v2](../theming/introduction.mdx) please refer to our [responsive layout guide](../code-examples/responsive-layout.mdx)

## Properties

### isOpen$

• **isOpen$**: `Observable`<`boolean`\>

Emits `true` if the channel list is in open state, otherwise it emits `false`

The `AutocompleteTextarea` component is used by the [`MessageInput`](./message-input.mdx) component to display the input HTML element where users can type their message.

Message send can be triggered with the `Enter` key, new line can be added with the `Shift+Enter` combination.

The `AutocompleteTextarea` extends the funcitonalities of the [`Textarea`](./textarea.mdx) component with autocomplete features:

- users can mention other users in a message

## Basic usage

You can use the `AutocompleteTextarea` component if you want to create your own message input component to override the default one:

```typescript
@Component({
  selector: "app-custom-message-input",
  template: `
    <stream-autocomplete-textarea
      [(value)]="textareaValue"
      (send)="messageSent()"
      (userMentions)="mentionedUsers = $event"
    ></stream-autocomplete-textarea>
    <!-- Other parts of the custom message input component -->
  `,
})
export class CustomMessageInputComponent {
  textareaValue = "";
  mentionedUsers = [];

  messageSent() {
    // Send your message
  }
}
```

## Inputs

### value

The value of the input HTML element.

| Type   | Default |
| ------ | ------- |
| string | ''      |

### areMentionsEnabled

If true, users can mention other users in messages. You also [need to use the `AutocompleteTextarea`](../concepts/opt-in-architecure.mdx) for this feature to work.

You can also set this input on the [`MessageInput`](./message-input.mdx/#inputs) component.

| Type    |
| ------- |
| boolean |

### mentionAutocompleteItemTemplate

You can provide your own template for the autocomplete list for user mentions. You also [need to use the `AutocompleteTextarea`](../concepts/opt-in-architecure.mdx) for this feature to work.

You can also set this input on the [`MessageInput`](./message-input.mdx/#inputs) component.

| Type                                                                                                                                                               |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| [`TemplateRef<MentionAutcompleteListItemContext>`](<(https://github.com/GetStream/stream-chat-angular/blob/master/projects/stream-chat-angular/src/lib/types.ts)>) |

### mentionScope

The scope for user mentions, either members of the current channel of members of the application.

You can also set this input on the [`MessageInput`](./message-input.mdx/#inputs) component.

| Type                       | Default   |
| -------------------------- | --------- |
| 'application' or 'channel' | 'channel' |

## Outputs

### valueChange

Emits the current value of the input element when a user types.

| Type                                                       |
| ---------------------------------------------------------- |
| [`EventEmitter`](https://angular.io/api/core/EventEmitter) |

### send

Emits when a user triggers a message send event (this happens when they hit the `Enter` key).

| Type                                                       |
| ---------------------------------------------------------- |
| [`EventEmitter`](https://angular.io/api/core/EventEmitter) |

### userMentions

Emits the array of users that are mentioned in the message, it is updated when a user mentions a new user or deletes a mention.

| Type                                                       |
| ---------------------------------------------------------- |
| [`EventEmitter`](https://angular.io/api/core/EventEmitter) |

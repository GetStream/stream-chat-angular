The `Channel` component is a container component that displays the [`ChannelHeader`](./channel-header.mdx), [`MessageList`](./message-list.mdx), [`NotificationList`](./notifications.mdx) and [`MessageInput`](./message-input.mdx) components. You can also provide the [`Thread`](./thread.mdx) component to use message [threads](https://getstream.io/chat/docs/javascript/threads/?language=javascript).

## Usage

```html
<div id="root">
  <stream-channel-list></stream-channel-list>
  <stream-channel>
    <!-- Main messages in the channel -->
    <stream-channel-header></stream-channel-header>
    <stream-message-list></stream-message-list>
    <stream-notification-list></stream-notification-list>
    <stream-message-input></stream-message-input>
    <!-- Main messages in the channel -->
    <!-- Thread messages in the channel -->
    <stream-thread name="thread">
      <stream-message-list
        name="thread-message-list"
        mode="thread"
      ></stream-message-list>
      <stream-message-input
        mode="thread"
        name="thread-message-input"
      ></stream-message-input>
    </stream-thread>
    <!-- Thread messages in the channel -->
  </stream-channel>
</div>
```

The `Channel` component doesn't do any settings, it's just displays its children components. The main purpose of this component is to provide the layout for the initial chat UI.

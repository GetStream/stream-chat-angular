The `Notification` component displays a notification in the [notification list](./NotificationListComponent.mdx).

## Basic Usage

```html
<stream-notification type="success">
  File upload was successfuly
</stream-notification>
```

## Customization

You can provide your own notification component by the [`CustomTemplatesService`](../services/CustomTemplatesService.mdx)
